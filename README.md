# Prompt Testing — Chatbot Educativo

## Integrantes
- Juan Rangel
- Santiago Villamizar


# Descripción

En esta actividad se realizó la configuración de Promptfoo junto con Gemini para probar prompts orientados a un chatbot educativo.

El objetivo fue analizar la calidad, claridad y consistencia de las respuestas generadas por la IA mediante diferentes pruebas automáticas.


# Herramientas utilizadas

- Node.js
- npm
- Promptfoo
- Gemini API
- Visual Studio Code


# Prompts utilizados

Se trabajó con prompts orientados a explicar temas educativos de forma sencilla y clara.

Ejemplos:

- Explicar temas académicos para personas sin conocimientos previos.
- Responder preguntas como un tutor amigable usando ejemplos simples.


# Assertions utilizadas

Durante las pruebas se utilizaron:

- icontains
- javascript
- llm-rubric

Estas assertions permitieron validar contenido, longitud y calidad de las respuestas.

---

# Evidencias

Se adjuntaron capturas de:
- instalación y configuración,
- ejecución en terminal,
- pruebas realizadas,
- resultados PASS y FAIL.


# Dificultades encontradas

Durante la actividad hubo problemas relacionados con:
- lentitud en las respuestas de Gemini,
- sobrecarga del servicio,
- errores durante algunos intentos de ejecución.


# Aprendizajes

Esta actividad ayudó a entender cómo funcionan las pruebas de prompts y cómo evaluar respuestas generadas por IA utilizando Promptfoo y Gemini.

También permitió observar cómo pequeños cambios en los prompts pueden modificar el comportamiento de la IA.


# Conclusión

La actividad permitió aplicar conceptos de Prompt Testing y QA para IA mediante pruebas automáticas, analizando tanto resultados correctos como errores durante la ejecución.
